Classic Minesweeper

Built with
- python 2.7
- pygame 1.9.2b8

Three levels of difficulty:
- Easy: 5x5, 5 mines
- Medium: 10x10, 25 mines
- Hard: 15x15, 45 mines

How to Play:
- The game follows classic minesweeper rules
- Use left mouse button to reveal cell
- Use right mouse button to mark/unmark cells
